# Env Variables

```
  NODE_ENV=development
  PORT=3000
  MONGO_URL=
  JWT_SECRET=
  PAYPAL_CLIENT_ID=
  PAYPAL_APP_SECRET=
  PAYPAL_API_URL=https://api-m.sandbox.paypal.com
  PAGINATION_LIMIT= 10
  CLOUDINARY_CLOUD_NAME=
  CLOUDINARY_KEY=
  CLOUDINARY_SECRET=
```

# Install Dependencies

```
  npm install
  cd frontend
  npm install
```

# Run

```
  # Run frontend & backend
  npm run dev

  # Run backend only
  npm run server
```

# Build & Deploy

```
  # Create frontend prod build
  cd frontend
  npm run build
```

# Seed Database

```
  # Import data - backend/users.js and backend/products.js
  npm run data:import

  # Destroy data
  npm run data:destroy
```
